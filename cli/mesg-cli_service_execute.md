## Usage
```
$ mesg-cli service:execute SERVICE TASK
```

### Aliases
```
$ mesg-cli service:exec
```

### Arguments
```
SERVICE  Hash or Sid
TASK     Task key
```

### Options
```
-d, --data=FOO=BAR  data required to run the task
-h, --help          show CLI help
-j, --json=json     Path to a JSON file containing the data required to run the task
-q, --quiet
--silent
```
