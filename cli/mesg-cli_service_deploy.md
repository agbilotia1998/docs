Deploy a service

## Usage
```
$ mesg-cli service:deploy [SERVICE_PATH_OR_URL]
```

### Arguments
```
SERVICE_PATH_OR_URL  [default: ./] Path of the service or url to access it
```

### Options
```
-h, --help     show CLI help
-q, --quiet
--env=FOO=BAR  set env defined in mesg.yml (configuration.env)
--silent
```
