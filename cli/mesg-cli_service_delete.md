Delete one or many services

## Usage
```
$ mesg-cli service:delete SERVICE
```

### Aliases
```
$ mesg-cli service:rm
$ mesg-cli service:destroy
```

### Arguments
```
SERVICE  Hash or Sid
```

### Options
```
-h, --help   show CLI help
-q, --quiet
--confirm    Confirm delete
--keep-data  Do not delete services' persistent data
--silent
```
